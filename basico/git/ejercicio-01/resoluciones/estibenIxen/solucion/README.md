# comandos de git.

git init para iniciar el repositorio
git pull para llamr a los datos almacenados de el repositosrio
git branch para crear las ramas
git status: para poder ver el estado de los cambios.
git branch -a: para aver todas las ramas
git commit -m para realizar mis commits.
git log para poder ver el historial de mis commits realizados
git checkout para cambiar de ramas
git add .: para agregar lso cambios realizados
