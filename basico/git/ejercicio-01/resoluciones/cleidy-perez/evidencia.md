## Explicación

Trabajar de forma ordenada paso a paso, de que todo tiene que tener un lugar para aguardar los datos o los codigo a realizar, luego en donde lo quiero empezar entonces utilizar git init, comenzar agregar los documentos, archivos carpetas para construir mi proyecto, para aguardar lo realizado se debe aguardar en la nube entonces se usa git add y realizar un commit especifico lo que se realizó para que las personas que colaboren sepan lo que se realizó.
