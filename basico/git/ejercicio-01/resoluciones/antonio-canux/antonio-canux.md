# Antonio Canux

## Objetivo

Inicializar un repositorio para los equipos de Esports practicando los diferentes comandos dentro de la terminal, y el manejo de Git.

## Comandos utilizados

```bash
cd ..
mkdir equipo-sports
cd equipo-sports
git init
git add .
git status
git commit -m "feat: Repositorio inicial para los equipos Shooter"
git log --oneline
```
- **cd**: Este comando nos permite retrocer en la ruta de las carpetas.
- **mkdir**: Este comando nos permite crear una nueva carpeta o directorio.
- **git init**: Con este comando inicializamos el repositorio de Git.
- **git add**: Este comando nos ayuda a preparar los archivos proximos a hacer el commit
- **git status**: Con este comando lograremos visualizar el estado de las diferentes ramas.
- **git commit**: Nos ayuda a realizar el commit como tal.
- **git log --oneline**: Nos enlista los commits realizados y desplazarnos entre elllos.
