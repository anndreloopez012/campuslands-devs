# 🎮 Sistema de Ranking para Escuadras (Torneo)

## Solución
La solución a este problema fue hacer primero el array de objetos para luego hacer un recorrido de ese aray y tomar su longitud para poder venir y multiplicarlo por 3 para continuar con una validacion si un equipo quedaba en primer lugar se le daban automaticamente 20 puntos al segundo 14 y asi consecutivamente ademas de eso volvimos hacer otro recorrido en la nueva lista pero esta vez con los nuevos valores y solo imprimirlos.
---
## Explicación
Para resolver el ejercicio correctamente, decidí empezar creando el array para tener los datos listos y continuando con un recorrido hacia el array y poder tomar la nueva lista y poner los valores ademas de eso decidi crear una validacion con if y else porque sentia que era lo mas facil para poder darle los puntos a los equipos y por ultimo volver a recorrer el nuevo array para imprimirlo con los valores nuevos.
