🧠 ¿Para qué sirve cada carpeta? 
---

```
src/: Es el corazón de todo. Aquí va el código fuente, como los scripts para que el personaje camine, salte o ataque.

assets/: Es el almacén de los recursos del juego. Lo dividí en tres partes para tener orden:

audio/: Para la música de fondo y los efectos de sonido.

images/: Para los dibujos, sprites de personajes, texturas y el diseño de la interfaz.

maps/: Para los archivos de diseño de niveles, salas y mapas del mundo explorable.

docs/: Aquí guardamos los documentos de texto, ideas de la historia, misiones y las reglas del juego.

tests/: Sirve para guardar las pruebas del código y revisar que nada se rompa cuando agreguemos cosas nuevas.

Nota: Todas las carpetas tienen un archivo .gitkeep adentro porque Git no sube carpetas si están totalmente vacías.
