videojuegos-accion-aventura
└─ assets
|   ├─ audio/

|

|
├─ tests/
|

### Explicación
La carpeta principal sera videojuegos-accion-aventura en el cual dentro tentra otras tres subcarpetas las cuales seran assets, src y test, ademas del archivo `readme.md` (este archivo) en el cual esta detallada la información. Dentro de assets estarn las carpetas audio(solo para archivos mp3 u otros no visuales), images (sera la parte visual del usuario y UI) y maps (escenarios, niveles y json de terreno).
En src iran los archivos principales y en test iran los archivos de integracion
