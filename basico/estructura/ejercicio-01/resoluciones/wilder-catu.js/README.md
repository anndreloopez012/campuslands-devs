# Estructura del Proyecto

## Carpetas principales

### `src/`
Contiene el código fuente principal de la aplicación o proyecto.

### `assets/`
Guarda recursos estáticos utilizados por el proyecto.

#### `assets/images/`
Imágenes e iconos.

#### `assets/audio/`
Archivos de sonido, música o efectos.

#### `assets/maps/`
Mapas, datos gráficos o recursos relacionados.

### `docs/`
Documentación técnica, manuales y notas del proyecto.

### `tests/`
Pruebas unitarias, de integración o automatizadas.

---

## Archivos `.gitkeep`

Se agregan archivos `.gitkeep` en carpetas vacías para que Git pueda incluirlas en el repositorio.
