# Ejercicio NO.1

## ¿Para que sirve cada carpeta?

-1. assests: es donde se almacena recursos que se necesita en la página y esto se divide en subcarpetas:
    -audio: se almacena todos los audios que se necesiten en el proyecto o videos.
    -imagenes: se almacena todas las imagenes como logo, imagenes del productos o iconos.
    -maps: se almacena todos los datos que se realiza un funcion que toman un objeto complejo y lo limpian, que los convierte en base de datos o formato para tu app, por ejemplo: 
        fecha_nacimiento a fechaNacimiento.
    
    O cuando hay un erro en el navegador, este te diga exactament en qué linea de tu código original ocurriió el fallo y no en el código final transformado y amontonado.

-2. docs: es donde se almacena todos los documentos como:
    -Manual del usuario.
    -Diagramas estructurados.
    -Arquitectura.
    -Diseño.

-3. src: es donde se almacena diferentes archivos como reautilización de componentes como:
        -Botones.
        -LogginForm entre otros. 

    También se aguarda archivos que hace que la págian web tenga lógica como archivos de javascript y conexiones de APIs o bases de datos.

-4. tests: se almacena todas las pruebas que se han realizado de la página web por ejemplo si se filtra bien los datos o el calculo de una operación es correcto, si realmente aguarda los datos, se comporta bien o interactua bien con las Apis o para realizar una simulación de datos fiticios para comprobar el comportamiento de la página.
