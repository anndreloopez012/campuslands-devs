# 🛵 Flujo del Proceso de Reparación

Este documento detalla el ciclo de vida de una orden de trabajo dentro de nuestro taller mecánico.

## 🔄 Estados de la Orden

1. **📥 Recibida / Diagnóstico:** La moto ingresa al taller. Se registran los datos del cliente, de la moto y los síntomas reportados.
2. **🛠️ En Reparación:** El mecánico asignado acepta la orden, revisa el diagnóstico y comienza a trabajar en las fallas detectadas.
3. **🧪 Pruebas de Control:** Se realiza una prueba de ruta y de seguridad para asegurar que la moto quedó al 100%.
4. **✅ Lista para Entrega:** Se genera la liquidación de la orden (costo de mano de obra + repuestos usados).
5. **🎉 Entregada:** El cliente paga, se le entrega su moto y se cierra la orden con éxito.
