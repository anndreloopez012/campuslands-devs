**Nombre:** Sergio Aju  
**Fecha:** 10 de junio de 2026

## 1. Descripción del problema
Hacer un ejercicio demostrando el flujo que tiene y la creacion de estructura y como se relaciona el html, css y javascript.

## 2. Estructura de archivos
La solución se organizó de la siguiente manera:
- `index.html`: Estructura principal del ranking.
- `css/style.css`: Estilos visuales del proyecto.
- `js/app.js`: Lógica encargada de manipular los datos.
- `data/players.json`: Archivo de datos en formato JSON con la información de los jugadores.
